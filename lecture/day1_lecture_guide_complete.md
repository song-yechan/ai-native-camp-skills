### Part A: AI 시대의 변화 (25분)

#### 슬라이드 A-1: Knowledge Worker
```
PART A

Knowledge Worker

```

**발표자 노트**:
- 피터 드러커가 1959년에 만든 개념
- 육체 노동 → 지식 노동으로 경제의 중심이 이동
- "생각하는 것이 곧 일인 사람들"

#### 슬라이드 A-2: 지식노동자의 능력
```
PART A

지식노동자의 능력

① 패턴 인식      데이터에서 의미를 읽어냄
② 맥락적 사고    상황에 맞는 판단을 내림
③ 메타인지       자기 사고 과정을 점검하고 교정

+ 지능형 메모리의 합성 능력
  200,000:1의 정보 압축과 전략적 제거
  멘탈 모델을 통한 체계적 의사결정

60년간 이것이 지식노동자의 경쟁력이었습니다.
```

**발표자 노트**:
- 패턴 인식: 수많은 데이터에서 의미 있는 것을 찾아내는 능력
- 맥락적 사고: 상황에 맞게 유연하게 판단하는 능력
- 메타인지: 내가 뭘 모르는지 아는 것, 내 사고를 점검하는 능력
- 200,000:1 압축: 읽은 책, 경험을 핵심만 남기고 압축하는 능력

#### 슬라이드 A-3: 2024년, AI가 이 영역에 들어왔습니다
```
PART A

2024년, AI가 이 영역에 들어왔습니다.

패턴 인식      AI가 인간보다 빠르고 정확
맥락적 사고    AI가 빠르게 따라오는 중
메타인지       아직 인간의 영역... 하지만 좁혀지는 중

→ 지능형 메모리   AI의 학습 데이터 = 전 인류의 경험
→ 정보 압축      AI의 attention mechanism
→ 멘탈 모델      AI에게 프레임워크를 주면 바로 적용

"대체되는 게 아니라, 증폭되는 겁니다."
```

**발표자 노트**:
- 핵심 메시지: "대체"가 아니라 "증폭"
- AI가 지식노동자의 능력을 빼앗는 게 아니라, 확장시켜줌
- 60년간의 경쟁력이 사라지는 게 아니라, 레버리지가 되는 것

#### 슬라이드 A-4: 도구의 진화 — 추상화 계층의 상승
```
PART A

도구의 진화 — 추상화 계층의 상승

1950s   기계어       0과 1로 직접 대화
1960s   어셈블리어   기호로 한 단계 추상화
1970s   고급 언어    인간의 논리에 가까운 문법
1990s   API          기능 단위로 호출
2010s   SaaS         클릭으로 소프트웨어 사용
2020s   자연어       말로 시키는 시대

↑ 추상화 계층이 올라갈수록
  더 많은 사람이 컴퓨터의 힘을 쓸 수 있게 되었습니다.
```

**발표자 노트**:
- 추상화 = 복잡한 것을 단순하게 감싸는 것
- 기계어 시절에는 컴퓨터 전문가만 사용 가능
- SaaS 시대에는 클릭만으로 누구나 사용 가능
- 자연어 시대에는 말만 하면 됨

#### 슬라이드 A-5: 추상화 계층의 상승 = 진입 장벽의 하락
```
PART A

추상화 계층의 상승 = 진입 장벽의 하락

기계어         수백 명
어셈블리       수천 명
고급 언어      수백만 명
API / SaaS     수십억 명
자연어         생각할 수 있는 모든 사람

로버트 라이시의 '상징 분석가' 4대 도구:
추상화 · 시스템적 사고 · 실험 · 협력

→ 이 도구들이 이제 자연어로 작동합니다.
```

**발표자 노트**:
- 도구가 진화할수록 사용할 수 있는 사람이 기하급수적으로 증가
- 로버트 라이시: 미래학자, "상징 분석가"가 미래 경제의 핵심이라고 예측
- 상징 분석가의 도구가 이제 자연어로 사용 가능

#### 슬라이드 A-6: Allocation Economy
```
PART A

Allocation Economy
할당 경제

       과거                    현재
"얼마나 잘 하는가"    →    "무엇을 시킬 것인가"
    = 생산성                  = 생산성
```

**발표자 노트**:
- 과거: 내가 얼마나 빨리, 잘 처리하느냐가 경쟁력
- 현재: 무엇을 AI에게 시키고, 내가 무엇에 집중할지 결정하는 것이 경쟁력
- "할당"의 시대 — 자원(AI)을 어디에 배분할지 결정하는 것

#### 슬라이드 A-7: 지식노동자의 새로운 핵심 역량

PART A

지식노동자의 새로운 핵심 역량

┌─────────────────────────────────────────────────────┐
│  과거 핵심 역량                                     │
│  패턴 인식 → 맥락적 사고 → 메타인지                 │
│                    ↓                                │

**발표자 노트**:

## 3. 핵심 개념 상세 정리

### 3.1 지식노동자(Knowledge Worker)란?

**정의**: "생계를 위해 생각하는 사람" (Peter Drucker, 1959)

**60년간의 핵심 역량**:
1. **패턴 인식**: 데이터에서 의미를 읽어냄
2. **맥락적 사고**: 상황에 맞는 판단을 내림
3. **메타인지**: 자기 사고 과정을 점검하고 교정

**+ 지능형 메모리의 합성 능력**:
- 200,000:1의 정보 압축과 전략적 제거
- 멘탈 모델을 통한 체계적 의사결정

### 3.2 AI 시대의 변화

**2024년, AI가 이 영역에 들어왔습니다**:

| 역량 | AI의 수준 |
|------|----------|
| 패턴 인식 | AI가 인간보다 빠르고 정확 |
| 맥락적 사고 | AI가 빠르게 따라오는 중 |
| 메타인지 | 아직 인간의 영역... 하지만 좁혀지는 중 |

**핵심 메시지**: "대체되는 게 아니라, 증폭되는 겁니다."

### 3.3 도구의 진화 — 추상화 계층의 상승

| 시대 | 도구 | 사용 가능 인원 |
|------|------|--------------|
| 1950s | 기계어 | 수백 명 |
| 1960s | 어셈블리 | 수천 명 |
| 1970s | 고급 언어 | 수백만 명 |
| 1990s~2010s | API / SaaS | 수십억 명 |
| 2020s | 자연어 | 생각할 수 있는 모든 사람 |

**로버트 라이시의 '상징 분석가' 4대 도구**: 추상화, 시스템적 사고, 실험, 협력
→ 이 도구들이 이제 자연어로 작동합니다.

### 3.4 Allocation Economy (할당 경제)

| 과거 | 현재 |
|------|------|
| "얼마나 잘 하는가" = 생산성 | "무엇을 시킬 것인가" = 생산성 |

### 3.5 지식노동자의 새로운 핵심 역량

1. **맥락 정의**: 내 일의 본질을 AI가 이해할 수 있도록 구조화 → **CLAUDE.md**
2. **판단 기준 설정**: 멘탈 모델을 AI에게 이식하여 프레임워크 제공 → **Skill**
3. **결과 검증**: 메타인지로 AI 산출물의 질을 통제

### 3.6 지식노동자 능력 × AI 증폭 = AI Native 워크플로우

| 지식노동자 능력 | × | AI의 증폭 |
|---------------|---|----------|
| 패턴 인식 | × | 무한한 데이터 처리 |
| 맥락적 사고 | × | 전체 파일 맥락 이해 |
| 메타인지 | × | 자율적 검증과 수정 |
| 전략적 직관 | × | 전 인류 경험의 합성 |
| 정보 압축 | × | 핵심 추출 자동화 |
| 멘탈 모델 | × | 프레임워크 즉시 적용 |

### 3.7 7개 핵심 기능 비유 정리

| 기능 | 비유 | 한 줄 설명 |
|------|------|----------|
| CLAUDE.md | 신입사원 업무 매뉴얼 | "우리 팀은 이렇게 일해" |
| Skill | 요리 레시피 | 재료와 순서만 따라하면 같은 결과 |
| MCP | USB-C 포트 | 외부 기기를 연결하는 통로 |
| Subagent | 부하 직원 | "이거 해놓고 끝나면 알려줘" |
| Agent Teams | 프로젝트 TF | 팀원들이 서로 소통하며 협업 |
| Hook | IFTTT | "이런 일이 생기면 자동으로 이거 해" |
| Plugin | 앱스토어 | 한 줄로 모든 설정 설치 |
