# AI Native Camp 2기 — 참가자 시작 가이드

## 사전 준비 (캠프 시작 전)

### 1. Claude Code 설치

```bash
npm install -g @anthropic-ai/claude-code
```

설치 확인:
```bash
claude --version
```

> Node.js가 없다면 먼저 설치: https://nodejs.org (LTS 버전 권장)

### 2. 작업 폴더 준비

```bash
mkdir ~/ai-camp && cd ~/ai-camp
```

### 3. 캠프 스킬 설치

```bash
git clone https://github.com/song-yechan/ai-native-camp.git ~/ai-native-camp-skills
ln -sf ~/ai-native-camp-skills/.claude/skills ~/.claude/skills
```

> 이렇게 하면 `/day1`, `/day2`, `/day3`, `/day4` 명령어가 활성화됩니다.

### 4. Claude Code 시작

```bash
claude
```

## 수업 흐름 (모든 Day 동일)

```
┌─ Phase A ─────────────────────────┐
│ Claude가 개념을 설명한다            │
│ 실습 내용을 안내한다                │
│ "직접 실행해보세요" → STOP          │
└───────────────────────────────────┘
       ↓ 직접 실행한 후
       ↓ "완료" 또는 "다음" 입력
┌─ Phase B ─────────────────────────┐
│ 퀴즈 1~3문제 출제                   │
│ 정답/오답 피드백                    │
│ 다음 블록으로 이동                  │
└───────────────────────────────────┘
```

### 사용 가능한 명령어

| 입력 | 동작 |
|------|------|
| `완료` / `다음` | 다음 Phase 또는 다음 Block으로 이동 |
| `skip` | 현재 블록 건너뛰기 |
| `Block 3` / `블록 3` | 특정 블록으로 바로 이동 |

## 자주 묻는 질문

**Q: 중간에 Claude Code를 껐다 켜면?**
A: `/day1`(또는 해당 Day) 다시 입력하면 시작 메뉴가 나옵니다. 이어서 할 블록을 선택하세요.

**Q: 실습이 안 되면?**
A: "완료"를 입력하고 다음 블록으로 넘어가면 됩니다. 실습 실패가 전체 진행을 막지 않습니다.

**Q: 퀴즈를 틀리면?**
A: 정답과 해설이 바로 제공됩니다. 점수와 무관하게 다음 블록으로 넘어갈 수 있습니다.

**Q: 터미널이 처음이에요**
A: Day 1 Block 4에서 필수 명령어 5개를 배웁니다. 모든 실습은 Claude에게 말로 시켜도 됩니다.
