## EXPLAIN

| 항목 | 내용 |
|------|------|
| 비유 | 부하 직원에게 위임 — "이거 조사해와" |
| 핵심 | Claude가 다른 Claude를 불러서 일을 시키는 것 |
| 장점 | 여러 일을 **동시에** 처리 (병렬 처리) |

### 비유: 팀장의 업무 분배

```
나 (팀장)
  │
  ├── Subagent 1: "Slack 메시지 요약해와"     ← 동시에 실행
  ├── Subagent 2: "이메일 정리해와"            ← 동시에 실행
  └── Subagent 3: "캘린더 확인해와"            ← 동시에 실행
  │
  ▼
  결과 합치기
```

- 순차 처리: 3개 × 30초 = **90초**
- 병렬 처리 (Subagent): 3개 동시 = **30초**

### 언제 사용하나?

| 상황 | Subagent 사용 |
|------|--------------|
| 독립적인 작업 여러 개 | ✅ |
| 서로 의존하는 순차 작업 | ❌ |
| 대량 데이터 처리 | ✅ |
| 간단한 단일 작업 | ❌ (오버헤드) |
